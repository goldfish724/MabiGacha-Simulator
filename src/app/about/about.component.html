Mabinogi Gachapon Simulator.

Hey fellow Milletians!

We bring you a tool to satisfy all your gachapon pangs and cravings without the cost and the in-game rewards...
Anyways, MabiGacha supports the following features thus far:

  • Saving a list of your loot (You'll lose it all if you clear your cookies, oh no!)
  • Keeping track of your expenditures
  • Single Gaching
  • Bulk Gaching
  • Continual Gaching until specified item is found

In the unfortunate event that you a roll an ultra rare drop in the simulator, I can only offer you my condolesences.

For those concerned about the accuracy of this simulator, Mabinogi provides the item table and percentage
for every gachapon, so rest assured! We're getting it straight from the source, beep boop.

QUESTION: How do we use the rates to accurately create a gachapon?

For the programmers of Erinn, check out the source code. Otherwise, Rates provided by Nexon are given in the format #.##%. For example, 0.16%, which when converted to decimal is 0.016 or 16/1000. wow math! Anyways, if you pretend gachapons are just a bunch of raffle tickets, imagine that we created a thousand raffle tickets in which only 16 of them have the name of that gacha item written on it. Apply this concept to the rest of the items to make up all one thousand raffles, and we simply draw tickets out of this raffle pool when you gach at random, read it, and put it back inside for the next round.
